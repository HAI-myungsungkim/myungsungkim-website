🎨 이미지 업로드 방법

옵션 1: Imgur (추천 - 가장 쉬움)
1. imgur.com/upload 접속
2. 이미지 드래그 앤 드롭
3. 우클릭 → "이미지 주소 복사"
4. 여기에 붙여넣기

옵션 2: 프로젝트 폴더
1. 프로젝트에 images/ 폴더 생성
2. 이미지 저장 (예: images/profile.jpg)
3. 상대 경로 입력: images/profile.jpg
4. GitHub에 이미지도 함께 커밋
